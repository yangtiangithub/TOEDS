# light-doctor

## Rrequirement
* [python3](https://www.python.org/downloads/)

## Usage

輸入 Symptoms set 中的症狀，1表示出現，-1表示未出現。

```
python3 diagnose.py 症狀A#1,症狀B#1,症狀C#1......
```

### Example

```
$ python3 diagnose.py 頭痛#1,牙痛#1
有吐嗎？(1 or -1) -1
你有發燒嗎？(1 or -1) 1
有拉肚子嗎？(1 or -1) -1
你有哪裡出血嗎？(1 or -1) -1
會覺得噁心嗎？(1 or -1) -1
你有乏力的感覺嗎？(1 or -1) -1
有便秘嗎？(1 or -1) -1
你有貧血嗎？(1 or -1) -1
目前看起來像是鼻竇炎，建議到耳鼻喉科進一步檢查
```
